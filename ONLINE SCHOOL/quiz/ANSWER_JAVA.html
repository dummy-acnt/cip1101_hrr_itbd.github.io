CORRECT ANSWERS FOR JAVA QUIZ

1. __________ keyword is used to create Package in Java.

  • Packs
  • Class
  • None
  • Interface

The correct answer is None.

2. The TreeMap class is used to implement

  • Map Interface
  • NavigableMap
  • AbstractMap Class
  • All of the above

The correct answer is All of the above.

3. Which inheritance is achieved using interface in Java?

  • Multilevel Inheritance
  • Single Inheritance
  • Multiple Inheritance
  • None of the above

The correct answer is Multiple Inheritance.

4. Methods of the interface must not be in nature.

  • Static
  • Final
  • Native
  • All of the above

The correct answer is All of the above.

5. How many classes we are able to extend in Java?

  • Only one class
  • Two class
  • Three class
  • Four class

The correct answer is Only one class.

6. Which keyword is used for using interface in class?

  • Throw
  • Extend
  • Implements
  • None of the above

The correct answer is Implements.

7. Which terms are included inside the package?

  • Method
  • Classes
  • Interfaces
  • All of the above

The correct answer is All of the above.

8. Which of the following package are built-in packages?

  • Uti
  • Java
  • Lang
  • All of the above

The correct answer is All of the above.

9. A package can be renamed without renaming the directory in which the classes are stored.

  • True
  • False

The correct answer is False.

10. Which of these can be used to fully abstract a class from its implementation?

  • Objects
  • None of the mentioned
  • Interfaces
  • Packages

The correct answer is Interfaces.
