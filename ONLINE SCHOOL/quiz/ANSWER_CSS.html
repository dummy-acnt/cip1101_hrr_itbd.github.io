CORRECT ANSWERS FOR CSS QUIZ

1. What does CSS stand for?

  • Creative Style Sheets
  • Colorful Style Sheets
  • Cascading Style Sheets

The correct answer is Cascading Style Sheets.

2. Where in an HTML document is the correct place to refer to an external style sheet?

  • In the ＜head＞ section
  • In the ＜body＞ section
  • At the end of the document

The correct answer is In the ＜head＞ section.

3. Which HTML tag is used to define an internal style sheet?

  • ＜script＞
  • ＜css＞
  • ＜style＞

The correct answer is ＜style＞.

4. How do you insert a comment in a CSS file?

  • 'this is a comment
  • //this is a comment //
  • /* this is a comment */

The correct answer is /* this is a comment */.

5. Which property is used to change the background color?

  • background-color
  • bgcolor
  • color

The correct answer is background-color.

6. Which CSS property is used to change the text color of an element?

  • fgcolor
  • color
  • text-color

The correct answer is text-color .

7. Which CSS property controls the text size?

  • text-style
  • font-size
  • font-style
  • text-size

The correct answer is font-size.

8. How do you make the text bold?

  • font-weight:bold;
  • font:bold;
  • style:bold;

The correct answer is font-weight:bold;.

9. Which property is used to change the left margin of an element?

  • padding-left
  • margin-left
  • indent

The correct answer is margin-left.

10. When using the padding property; are you allowed to use negative values?

  • Yes
  • No

The correct answer is No.
