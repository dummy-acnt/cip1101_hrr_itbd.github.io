1. Inside which HTML element do we put the JavaScript?

The correct answer is ＜script＞.

  • The ＜head＞ section
  • The ＜body＞ section

  • None of the above

The correct answer is Both the ＜head＞ section and the ＜body＞ section are correct.

3. What is the correct syntax for referring to an external script called "xxx.js"?

  • ＜script href="xxx.js"＞
  • ＜script src="xxx.js"＞

The correct answer is ＜script src="xxx.js"＞.

4. The external JavaScript file must contain the ＜script＞ tag.

  • True

The correct answer is False.

5. How do you write "Hello World" in an alert box?

  • msgBox("Hello World");
  • msg("Hello World");
  • alert("Hello World");
  • alertBox("HEllo World");

The correct answer is alert("Hello World");.

6. How do you call a function named "myFunction"?

  • call function myFunction()
  • call myFunction()
  • myFunction()

The correct answer is myFunction().

7. How can you add a comment in a JavaScript?

  • //This is a comment
  • 'This is a comment

The correct answer is //This is a comment.

8. How do you round the number 7.25, to the nearest integer?

  • Math.rnd(7.25)
  • rnd(7.25)
  • round (7.25)
  • Math.round(7.25)

The correct answer is Math.round(7.25).

9. How do you find the number with the highest value of x and y?

  • Math.ceil(x,y)
  • top(x,y)

The correct answer is Math.max(x,y).
