CORRECT ANSWERS FOR PYTHON QUIZ

1. What is a correct syntax to output "Hello World" in Python?

  • echo "Hello World"
  • print("Hello World")
  • p("Hello World")

The correct answer is print("Hello World").

2. How do you insert COMMENTS in Python code?

  • //This is comment
  • #This is comment
  • /*This is a comment*/

The correct answer is #This is comment.

3. Which one is NOT a legal variable name?

  • my-var
  • _myvar
  • Myvar
  • my_var

The correct answer is my-var.

4. How do you create a variable with the numeric value 5?

  • x = 5
  • x = int(5)
  • Both the other answers are correct

The correct answer is x = 5.

5. What is the correct file extension for Python files?

  • .pyt
  • .pt
  • .py
  • .pyth

The correct answer is .py.

6. In Python, 'Hello', is the same as "Hello"

  • True
  • False

The correct answer is True.

7. Which method can be used to remove any whitespace from both the beginning and the end of a string?

  • len()
  • strip()
  • ptrim()
  • trim()

The correct answer is trim().

8. Which method can be used to return a string in upper case letters?

  • uppercase()
  • upper()
  • toUpperCase()
  • upperCase()

The correct answer is toUpperCase().

9. Which operator is used to multiply numbers?

  • #
  • %
  • x
  • *

The correct answer is *.

10. Which operator can be used to compare two values?

  • ＜＞
  • =
  • ==
  • ＞＜

The correct answer is ==.
