CORRECT ANSWERS FOR DATA STRUCTURES QUIZ

1. Example of linear data structure except

  • array
  • tree
  • queue
  • stack

The correct answer is tree.

2. _________ is a collection of elements used to store the same type of data.

  • Array
  • Switch
  • Case
  • Loop

The correct answer is Array.

3. Which of the following is NOT a Type of Data Structures?

  • Array
  • Tables
  • Stacks
  • Trees

The correct answer is Tables.

4. Storage for data defined in terms of set of operations to be performed on the data

  • Arrays
  • Variables
  • Abstract Data Types
  • Algorithms

The correct answer is Abstract Data Types.

5. It use pointer to link nodes.

  • Variable
  • Link List
  • Pointer
  • Trees

The correct answer is Link List.

6. LIFO stands for

  • List of Outputs
  • Last in First Out
  • First in Last Out
  • None of them

The correct answer is Last in First Out.

7. Act of adding values into a stack is called

  • Popping
  • Polling
  • Pushing
  • None

The correct answer is Pushing.

8. A stack overflow is an undesirable condition in which a particular computer program tries to use more
     memory space than the called stack has available.

  • True
  • False

The correct answer is True.

9. For a linked list, what meta-data will need to be stored with each data entry?

  • The Name of the list
  • The address of the current data entry
  • The address of the next data entry in the list
  • The meaning of life

The correct answer is The address of the next data entry in the list.

10. What would a dynamic data structure entail?

  • Memory is allocated to the data structure as the program executes.
  • Memory is allocated to the data structure at compile time.
  • The size of the structure is fixed.
  • The data structure can also double as TNT.

The correct answer is Memory is allocated to the data structure as the program executes.
