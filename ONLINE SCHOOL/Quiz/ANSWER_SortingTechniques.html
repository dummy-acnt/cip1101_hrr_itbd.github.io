CORRECT ANSWERS FOR SORTING TECHNIQUES QUIZ

1. What does a sorting algorithm do?

  • Finds an item of data in a set of data
  • Saves a set of data
  • Puts a list of items into order

The correct answer is Puts a list of items into order.

2. What does a bubble sort do?

  • Sorts a list by comparing two items that are side by side, to see which is out of order
  • Separates a list of data into different collections of data, before sorting and gathering back into a list
  • Finds an item of data in a list

The correct answer is Sorts a list by comparing two items that are side by side, to see which is out of order.

3. How many passes will a bubble sort go through?

  • Only one pass
  • Two passes
  • Several passes - until the data is fully ordered

The correct answer is Several passes - until the data is fully ordered.

4. Why does a bubble sort do a final pass even when the data is in the correct order?

  • To save the data
  • It does not recognise that the data is in order until the final pass requires no changes
  • It needs to do this to put the data back in to a list

The correct answer is It does not recognise that the data is in order until the final pass requires no changes.

5. Which of the following is not a type of sorting algorithm?

  • Merge sort
  • Insertion sort
  • Long sort

The correct answer is Long sort.

6. Which of the following is an advantage of a bubble sort?

  • It is a very small and simple computer program
  • It takes a very long time to run
  • It is very quick

The correct answer is It is a very small and simple computer program.

7. The Selection Sort algorithm repeatedly finds the smallest value in the list, then swaps it into its
    sorted position.

  • True
  • False

The correct answer is True.

8. Why might a sorting algorithm be needed before a search?

  • Data is less when sorted
  • Data cannot be searched if it not sorted
  • Data can be retrieved faster and easier when sorted
  • Data must always be sorted before it is searched

The correct answer is Data can be retrieved faster and easier when sorted.

9. Repeatedly compare neighbor pairs and swap if necessary

  • Bubble Sort
  • Insertion Sort
  • Selection Sort

The correct answer is Bubble Sort.

10. Insertion Sort is an algorithm that selects the smallest element from an unsorted list in each iteration
     and places that element at the beginning of the unsorted list.

  • True
  • False

The correct answer is False.
