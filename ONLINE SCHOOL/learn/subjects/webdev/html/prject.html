What is HTML?



HTML is the standard markup language for creating Web pages.




  • HTML stands for Hyper Text Markup Language
  • HTML describes the structure of a Web page
  • HTML allows users to create and structure sections, headings, links, paragraphs, and more, on a website using various tags and elements. Almost everything you want to create on a web page can be done using a specific HTML code.


    • HTML BASIC FORMAT PAGE STRUCTURE


  • <!Doctype html> A doctype declaration, also known as a document type declaration, informs the web browser about the markup language used to create the current page. It's not a tag or an element. There is no case distinction in the doctype statement.

  • <html>This tag is used to specify the HTML document's root element. This element informs the browser that the document is HTML. It's the second outer container element, and all other elements are contained within it.

  • <head>This element is used to establish the HTML document's head section, which contains information about the document. On the front end of a website, elements within the head tag are not visible.

  • <body>The body element is used to encompass all of a webpage's viewable content. In other words, the body content is what appears on the front end of the browser.
