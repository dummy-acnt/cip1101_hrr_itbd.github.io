CSS BACKGROUND



Background effects for elements are defined using CSS background properties. There are lots of properties to design the background.




CSS BACKGROUND PROPERTIES:

  • CSS Background-color Property: In CSS, the background-color property is used to specify an element's background color.

  • CSS Background-image Property: The background-image property is used to give an element one or more background pictures.

  • CSS Background-repeat Property: used to repeat the background picture both horizontally and vertically.


  • CSS Background-attachment Property:used to describe the type of attachment of the background picture to its container in CSS.


  • CSS CSS Background-position Property: CSS body-position property is mostly used to place an image.
