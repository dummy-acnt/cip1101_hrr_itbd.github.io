What is CSS?



CSS (Cascading Style Sheets) is a stylesheet language for creating appealing webpages. The goal is to make the process of creating web pages attractive as simple as possible. You may use it to style web pages. More crucially, it allows you to do it without having to worry about the HTML that each web page contains.




Three types of css:

  • Inline
  • Internal or Embeded
  • External

    CSS SYNTAX

    A CSS file contains style rules that the browser interprets and then applies to the components in your document. A selector and declaration block make up a style rule set.



    • The selector identifies the HTML element to be styled.
    • One or more declarations are separated by semicolons in the declaration block.
    • A semicolon always terminates a CSS declaration, and declaration blocks are enclosed by curly braces.
    • A colon separates the CSS property name and value in each declaration.
      For example:
      –; property is color, and value is red.
      –; value is 15px, and property is font-size.
