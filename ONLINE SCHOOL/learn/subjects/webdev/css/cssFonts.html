CSS FONTS



The CSS font is used to set the font’s content of the HTML element.



CSS font-family

It is used to set the font type of an HTML element. It holds several font names as a fallback system.



CSS font-weight

It's used to change the font's boldness. Its value can be “normal, bold, lighter, bolder”.



CSS font-variant

It is used to create the small-caps effect. It can be “normal or small-caps”.



CSS font-size

It's used to change the size of an HTML element's font. Font size may be set in a variety of ways, including "pixels, %, em," or "small, big."
