CSS COMMENTS



In CSS, comments are statements in your code that the compiler ignores and does not execute. The code is explained via comments. They improve the readability and comprehension of the application.



A single-line or multi-line comment is acceptable. Both single and multiline comments can employ the /* */ comment syntax. Although older browsers support the <-- --> syntax for concealing in CSS, it is no longer supported. Adding comments to the code is a good practice that can assist someone comprehend it if they read it or revisit it later.

Note: The outputs are the same because comments are ignored by the browser and are not interpreted.
