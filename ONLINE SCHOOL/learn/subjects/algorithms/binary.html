Binary Search

Binary Search is an algorithm for searching through elements of a presorted array by repeatedly dividing the search interval in half.The idea of binary search is to use the information that the array is sorted and reduce the time complexity to O.

Example:

Step 1 - take the element in the center of an array which is 16

Step 2 - Compare 23 to 16. since 16 is less than 23, all the elements lower than 16 will be eliminated

Step 3 - take the element in the center the new array, this time it's 56. since 56 is more than 23, all the elements more than 56 will be eliminated.

Step 4 - take the element in the center of the remaining array. and this time it's 23. return the position of 23.
