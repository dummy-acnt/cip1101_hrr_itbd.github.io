Selection Sort

Selection sort is one algorithm used to sort a sequence of numbers by using a linear search, the smallest value in the sequence is located.

Selection sort selects the smallest element from an unsorted list in each iteration and places that element at the beginning of the unsorted list.

Example:

Step 1 - the smallest value swaps with the leftmost number and is considered fully sorted.

Step 2 - If the smallest value is already in the leftmost position, nothing will change.

Step 3 - The same operations are repeated until all of the elements are fully sorted.
